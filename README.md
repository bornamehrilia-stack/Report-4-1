# گزارش پروژه: ساخت کاراکترهای سفارشی روی LCD 16x2 با آردوینو

## ۱. مقدمه  
در این پروژه، به جای استفاده از کاراکترهای پیش‌فرض LCD (مثل حروف الفبا و اعداد)، **سه کاراکتر سفارشی** (قلب، زنگ و فضانورد) طراحی و روی LCD نمایش داده می‌شوند. 



## ۳. قطعات مورد استفاده  
- برد آردوینو Uno  
- نمایشگر LCD 16x2 (معمولاً با کنترلر HD44780)  
- یک پاتنسیومتر ۱۰k اهم (برای تنظیم کنتراست LCD)  
- برد برد (Breadboard)  
- سیم‌های جامپر (jumper wires)


## ۴. نحوه‌ی ساخت مدار  
- **LCD**:  
  - پین 1 (VSS) → GND  
  - پین 2 (VDD) → 5V  
  - پین 3 (VO) → وسط پاتنسیومتر (برای تنظیم کنتراست)  
  - پین 4 (RS) → پین ۱۲ آردوینو  
  - پین 5 (RW) → GND (در حالت نوشتن)  
  - پین 6 (E) → پین ۱۱ آردوینو  
  - پین 7-10 (D0-D3) → نادیده گرفته می‌شوند (در حالت 4-bit)  
  - پین 11 (D4) → پین ۵ آردوینو  
  - پین 12 (D5) → پین ۴ آردوینو  
  - پین 13 (D6) → پین ۳ آردوینو  
  - پین 14 (D7) → پین ۲ آردوینو  
  - پین 15 (A) → 5V (با مقاومت ۲۲۰ اهم برای روشن کردن پس‌زمینه)  
  - پین 16 (K) → GND  
.

## ۵. کد برنامه (اسکچ):

```cpp
#include <LiquidCrystal.h>   // اضافه کردن کتابخانه LCD

// ایجاد شیء LCD با پین‌های مشخص
LiquidCrystal lcd(12, 11, 5, 4, 3, 2);

// تعریف کاراکترهای سفارشی (هر کاراکتر 8 بایت = 8 ردیف، هر ردیف 5 بیت)
byte Heart[8] = {
  0b00000,
  0b01110,
  0b11111,
  0b11111,
  0b01110,
  0b00100,
  0b00000,
  0b00000
};

byte Bell[8] = {
  0b00100,
  0b01110,
  0b01110,
  0b11111,
  0b00000,
  0b00100,
  0b00000,
  0b00000
};

byte Alien[8] = {
  0b11111,
  0b10101,
  0b11111,
  0b11111,
  0b01110,
  0b01010,
  0b00000,
  0b00000
};

void setup() {
  lcd.begin(16, 2);   // شروع LCD با 16 ستون و 2 ردیف

  // ایجاد کاراکترهای سفارشی (ذخیره در حافظه LCD)
  lcd.createChar(0, Heart);   // کاراکتر 0 = قلب
  lcd.createChar(1, Bell);    // کاراکتر 1 = زنگ
  lcd.createChar(2, Alien);   // کاراکتر 2 = فضانورد

  lcd.clear();                // پاک کردن صفحه
  lcd.print("Custome Character");  // نمایش متن
}

void loop() {
  // نمایش کاراکترهای سفارشی در ردیف دوم
  lcd.setCursor(0, 1);        // قرار دادن کورسور در ابتدا ردیف دوم
  lcd.write(byte(0));         // نمایش کاراکتر 0 (قلب)

  lcd.setCursor(8, 1);        // قرار دادن کورسور در ستون ۸ ردیف دوم
  lcd.write(byte(1));         // نمایش کاراکتر 1 (زنگ)

  lcd.setCursor(14, 1);       // قرار دادن کورسور در ستون ۱۴ ردیف دوم
  lcd.write(byte(2));         // نمایش کاراکتر 2 (فضانورد)
}
```


## ۷. نتایج  
LCD به درستی کاراکترهای سفارشی را نمایش می‌دهد. این نشان‌دهنده‌ی عملکرد صحیح ارتباط آردوینو با LCD و تعریف کاراکترهای سفارشی است..
